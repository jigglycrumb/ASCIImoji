What is ASCIImoji?

Wouldn´t it be cool if you could get an emoticon like ʕ·͡ᴥ·ʔ just by typing (bear)?
Well, that's exactly what ASCIImoji does!

The latest version 1.6.0 has different ASCIImojis built in (see list below).

ASCIImoji is currently available in these formats:

Mac OS text shortcuts

If you use Mac OS, this is your best choice. After installation, you can use ASCIImoji system-wide.

Download Installation instructions »

Chrome extension

The Chrome extension enables ASCIImoji for websites. You can even define your own emoticons!
