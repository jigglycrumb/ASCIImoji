What is ASCIImoji?

ASCIImoji brings ASCII emoticons to the web.

Wouldn´t it be cool if you could get an emoticon like ʕ·͡ᴥ·ʔ﻿ just by typing (bear)?
ASCIImoji´s Chrome Extension lets you do just that, on any website you´re visiting. And unlike most other emoticon packages out there the recipient doesn´t need to install anything extra because the emoticons are just text.

The latest version (1.4.3) has different ASCIImojis built in (see list below). Whether you´re an end-user or a developer, the three available releases got you covered:

Chrome extension

The Chrome extension enables ASCIImoji support for all websites. If you want to use ASCIImoji in your daily browsing experience, install this extension. You can even define your own emoticons!

Install from Chrome Webstore »

jQuery plugin

The jQuery plugin makes it easy for developers to integrate ASCIImoji in their websites or web applications. If you´re developing something and already use jQuery, this is your choice.

View at Github »

Javascript source

The plain Javascript source is the foundation of the jQuery plugin and the Chrome extension. If you don´t want to include jQuery or use ASCIImoji only programmatically, this is what you´ll want.
