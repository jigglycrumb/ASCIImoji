What´s this good for?

ASCIImoji enables support for Unicode emoticons on every website your are visiting. To use them, just type the keywords as listed in the tables below into an web text box and ASCIImoji will convert it to the corresponding emoticon.

Settings

Settings are automatically saved when you change them.

Here you can set the prefix and suffix needed to trigger the emoticon conversion (see the "You type" column in the tables below)
When enabled, ASCIImoji will be added to Chrome's right-click menu on text inputs.

Your Custom Emoticons

Use the form below to add your own emoticons to the collection. You can also overwrite the built-ins by using keywords that are already used. Seperate multiple keywords with commas, like so: "hello,ohai,wave"

All the built-in emoticons, for your reference.
